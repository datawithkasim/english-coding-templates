🎢 005 Week 4 — Helper Functions 2 (Extension · Easy)

Topic: Petting Zoo — Helper Functions · Course: Theme Park Creator · Level: Easy · Time: about 40 minutes

You are building a little petting zoo: a fence around a square, a row of trees, and animals inside.

That is a lot of blocks. So you teach the computer a small job once, give it a name, then use it again and again. That named job is a helper function.

💡 The whole idea: write a job once → use it many times. Less typing. Fewer mistakes. Bigger builds.

Your four tools this week

blocks.place(OAK_FENCE, pos(0, 0, 0))   # put one block somewhere
mobs.spawn(CHICKEN, pos(3, 1, 3))       # put one animal somewhere
def post(x, z):                         # teach a job a name
for i in range(0, 6):                    # do the line below 6 times

2 · Predict 🔮

Read the code. Write what appears in the world, and why.

def post(x, z):
    blocks.place(OAK_FENCE, pos(x, 0, z))

You run only this. What builds?

💡 Writing def does not build anything. It only teaches the job. It builds when you call the name.

post(2, 2)
post(4, 4)

Now what builds, and where? How many posts?

for i in range(0, 6):
    post(i, 0)

How many posts, and in a line or a pile? Why?

4 · Build the Petting Zoo 🐑

Here is the whole build. Most is written for you. Fill the blanks ( ____ ), then type it and run it.

Part A — one fence post, taught once

def post(x, z):
    blocks.place(OAK_FENCE, pos(x, 0, z))

Part B — the whole fence (calls post four times each loop)

def build_fence():
    for i in range(0, 6):
        post(i, 0)     # front wall
        post(i, 5)     # back wall
        post(0, i)     # left wall
        post(____, i)  # right wall  <-- fill the blank

💡 One loop, four posts each time = the whole square. You wrote post once; it runs 24 times.

Part C — one tree, then a row of trees

def tree(x):
    blocks.place(OAK_LOG, pos(x, 0, 4))
    blocks.place(OAK_LOG, pos(x, 1, 4))
    blocks.place(OAK_LEAVES, pos(x, 2, 4))

def plant_trees():
    for x in range(1, 5):
        ____(x)        # <-- call your tree helper

Part D — one animal, then a flock

def animal(x):
    mobs.spawn(CHICKEN, pos(x, 1, 2))

def add_animals():
    for x in range(1, 5):
        animal(x)

Part E — the whole zoo, in three lines

build_fence()
plant_trees()
____()             # <-- add the animals

💡 Read Part E top to bottom — it reads like a sentence. Three helper names = a whole petting zoo. That is the power of helper functions.
