# 🎮 RB005 — 드림 게임 스튜디오

8주 동안 지금까지 배운 모든 것을 모아 **자기만의 오리지널 게임**을 기획하고, 만들고, 다듬어서 출시합니다. 마지막 주에는 직접 게임을 발표합니다.

로블록스 트랙의 **다섯 번째 코스 (졸업 작품)** 입니다. RB001~RB004를 마친 친구들이 듣습니다. 새 문법보다 **스스로 결정하는 힘**을 기르는 코스입니다 — 진짜 게임 스튜디오가 일하는 방식 그대로요.
