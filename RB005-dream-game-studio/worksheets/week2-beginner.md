# 🎮 RB005 Week 2 — English Worksheet (Beginner)

**Topic:** Greybox It — Find the Fun First · **Course:** Dream Game Studio · **Level:** Beginner · **Time:** about 30 minutes

This week you build your WHOLE map with plain gray parts. This is called **greyboxing** — find the fun first, decorate later.

---

## 2 · Find the Difference 🐛

Each pair shows a good plan first, then a broken version of the same idea. Circle what's different and write one short sentence about the problem.

**Pair A** — Every map needs a goal.

```
# good
START → bridge → jumps → GOAL flag
```

```
# broken
START → bridge → jumps → ???
```

**What is missing? How does the player win?**

```
# good
Whole map built with plain gray parts.
Colors come later.
```


## 4 · Tell Me What You Built 📸

Now open Roblox Studio and greybox your whole map — start, path, goal — all in plain gray parts. Playtest it for 10 minutes. When you finish, come back here.

Send a photo or video of your greybox map, then explain what you found. Use these sentence starters — write 2 or 3 sentences.

> My map starts at … and the goal is …
>
> The fun part was …
>
> The boring part was …

<div class="write-space tall" style="min-height: 240px"></div>

---

## 5 · Record Your Walkthrough 🎥

Take a video on your phone (or a parent's phone) while you walk through your greybox map. Talk like a tour guide. Try to use these words: **greybox**, **start**, **goal**, **playtest**.

> 1. Walk from the start to the goal like a player.
> 2. Point at the goal and say how you win.
> 3. Say one thing you will fix before decorating.

**Write what you will say in your video.** You can read from it while filming.
