## 2 · Find the Difference 🐛

Each pair shows the clean launch setup first, then a broken version of the same idea. Circle what's different and write one short sentence about the problem.

**Pair A** — Everyone in class should be able to join your game today.

```
-- clean
Game Settings → Permissions → Public
```

```
-- buggy
Game Settings → Permissions → Private
```

**What is wrong? What happens when your classmates try to join?**

<div class="write-space short"></div>

**Pair B** — Players should start with **0 coins** and earn them by playing.

```lua
-- clean
local coins = Instance.new("IntValue")
coins.Name = "Coins"
coins.Value = 0
```

```lua
-- buggy
local coins = Instance.new("IntValue")
coins.Name = "Coins"
coins.Value = 999
```

**What is wrong? Why is starting with 999 coins a problem on launch day?**
