# 🏭 RB003 — 타이쿤 엔지니어

8주 동안 드로퍼·컨베이어·업그레이드 버튼이 있는 **나만의 타이쿤 게임**을 만들고, 게임을 꺼도 돈이 사라지지 않게 **저장 기능**까지 넣어 출시합니다.

로블록스 트랙의 **세 번째 코스**입니다. RB001~RB002를 마친 친구들이 듣습니다. 배열·딕셔너리·데이터 저장을 배우는, 코드가 "진짜 소프트웨어"처럼 보이기 시작하는 코스입니다.
