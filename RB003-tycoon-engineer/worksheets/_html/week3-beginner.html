🏭 RB003 Week 3 — English Worksheet (Beginner)

Topic: Arrays — A List of Upgrades · Course: Tycoon Engineer · Level: Beginner · Time: about 30 minutes

This week your tycoon gets an array — one variable that holds a whole list of upgrades. You grab items with an index like upgrades[1].

1 · Predict 🔮

Read the script. Before you imagine running it in Roblox Studio, circle or write your answer.

local upgrades = {"Fast Dropper", "Gold Ore", "Mega Conveyor"}
print(upgrades[1])

Does it print Fast Dropper? Circle one: yes · no

Why?

local upgrades = {"Fast Dropper", "Gold Ore", "Mega Conveyor"}
print(upgrades[4])

There is no item 4. Does it print an upgrade name? Circle one: yes · no

2 · Find the Difference 🐛

Each pair shows clean code first, then a broken version of the same idea. Circle what's different and write one short sentence about the bug.

Pair A — The script should print the first upgrade. Luau lists start at 1.

-- clean
local upgrades = {"Fast Dropper", "Gold Ore", "Mega Conveyor"}
print(upgrades[1])

-- buggy
local upgrades = {"Fast Dropper", "Gold Ore", "Mega Conveyor"}
print(upgrades[0])

What is wrong? What does the buggy script print?

Pair B — A list needs commas between the items.

-- clean
local upgrades = {"Fast Dropper", "Gold Ore", "Mega Conveyor"}

-- buggy
local upgrades = {"Fast Dropper" "Gold Ore" "Mega Conveyor"}

What is wrong?
