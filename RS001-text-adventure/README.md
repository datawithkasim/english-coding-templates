# 📜 RS001 — 텍스트 어드벤처 게임 (Python)

**영어코딩** 16주 파이썬 과정입니다. 매주 한 가지 개념을 익히고, 그 위에 코드를 쌓아 올려 마지막에는 변수·리스트·조건문·함수·랜덤이 한 흐름으로 이어지는 **나만의 텍스트 어드벤처 게임**을 완성합니다. 처음 만든 코드는 마지막 주까지 버리지 않습니다.

한 슬라이드 덱을 **학습 주**(개념 도입)와 **응용 주**(깊이 활용 + 프로젝트 확장) 두 주에 걸쳐 사용합니다.
