# 🔺 001 — Pyramid Problems / 피라미드 문제

8주 동안 마인크래프트 에이전트로 첫 줄부터 다이아몬드 구조물까지 직접 만듭니다.
8 weeks of Minecraft Agent coding — from a single line of blocks to a full diamond.

## 📅 주차별 학습 / Weekly Plan

| 주 / Week | 주제 / Topic | 결과물 / Result |
|----------|-------------|----------------|
| 1 | 에이전트와 첫 인사 / Meeting the Agent | 7칸 직선 길 / 7-block path |
| 2 | 반복문 / Loops | 20블록 벽 / 20-block wall |

| 6 | 거꾸로 피라미드 / Inverted Pyramid | 거꾸로 피라미드 / Inverted pyramid |
