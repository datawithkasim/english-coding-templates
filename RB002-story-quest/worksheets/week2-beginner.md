# 📜 RB002 Week 2 — English Worksheet (Beginner)

**Topic:** Strings — Talking Code · **Course:** Story Quest · **Level:** Beginner · **Time:** about 30 minutes

This week your code starts **talking**. You join strings together with `..` to build messages.

## 2 · Find the Difference 🐛

Each pair shows clean code first, then a broken version of the same idea. Circle what's different and write one short sentence about the bug.

**Pair A** — The script should print `Hello, Mina!`.

```lua
-- clean
local name = "Mina"
print("Hello, " .. name)
```

```lua
-- buggy
local name = "Mina"
print("Hello, " + name)
```

**What is wrong? Which symbol joins strings?**

<div class="write-space short"></div>

**Pair B** — The script should print `Welcome to the village`.

```lua
-- clean
local place = "the village"
print("Welcome to " .. place)
```

```lua
-- buggy
local place = "the village"
print("Welcome to" .. place)
```

**What is wrong? Look inside the quotes very carefully.**

<div class="write-space short"></div>
