# 📜 RB002 — 스토리 퀘스트

8주 동안 말하는 NPC, 갈래 대화, 퀘스트 아이템이 있는 **나만의 모험 게임**을 만들어 로블록스에 출시합니다.

로블록스 트랙의 **두 번째 코스**입니다. RB001(점프맵 만들기)을 마친 친구들이 듣습니다. 대사와 문장을 코드로 다루는 코스라 영어 연습 효과가 특히 큽니다.
