# ⚔️ RB004 — 아레나 레전드

8주 동안 두 팀이 맞붙는 **라운드제 멀티플레이어 미니게임**을 만듭니다. 팀, 타이머, 공지, 파워업, 점수판까지 — 친구들과 같이 노는 진짜 멀티플레이어 게임입니다.

로블록스 트랙의 **네 번째 코스**입니다. RB001~RB003을 마친 친구들이 듣습니다. 서버와 클라이언트라는 개념을 처음으로 배우는 코스입니다.
