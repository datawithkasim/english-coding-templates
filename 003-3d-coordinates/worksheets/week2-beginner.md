# 🧊 M003 Week 2 — English Worksheet (Beginner)

**Topic:** Lines and Grids · **Course:** 3D Coordinates · **Level:** Beginner · **Time:** about 30 minutes

This week you place a **line of blocks with a loop**. One number stays the same, the other grows by 1 each time.

---

## 1 · Predict 🔮

Read the steps. Before you imagine it happening, circle or write your answer.

```
set z to 0
repeat 3 times:
    place stone block at (2, z)
    add 1 to z
```

**z grows by 1 each time. Do the blocks make a line? Circle one:** yes · no

**Why?**

## 2 · Find the Difference 🐛

Each pair shows clean steps first, then a broken version of the same idea. Circle what's different and write one short sentence about the bug.

**Pair A** — The loop should make a line. The blocks need to **move along** after each place.

```
# clean
set z to 0
repeat 4 times:
    place stone block at (3, z)
    add 1 to z
```

```
# buggy
set z to 0
repeat 4 times:
    place stone block at (3, z)
```

**What is wrong? Where do all the buggy blocks land?**

<div class="write-space short"></div>

**Pair B** — The line should be **4 blocks long**.

```
# clean
set x to 0
repeat 4 times:
    place gold block at (x, 0)
    add 1 to x
```

```
# buggy
set x to 0
repeat 3 times:
    place gold block at (x, 0)
    add 1 to x
```

**What is wrong?**

## 4 · Tell Me What You Built 📸

Now switch to your homework world. Use a loop to build a **line of blocks**, then try a second line to start a square. When you finish, come back here.

Send a photo or video of your lines, then explain what you did. Use these sentence starters — write 2 or 3 sentences.

> My loop repeated … times because …
>
> The number that grew was …
>
> One tricky moment was when …

## 5 · Record Your Walkthrough 🎥

Take a video on your phone (or a parent's phone) while you show your line in the world. Talk like you are teaching a friend. Try to use these words: **loop**, **line**, **repeat**, **grow**.

> 1. Show your line, then show the code that built it.
> 2. Read your loop out loud and say which number grows.
> 3. Say in your own words why a loop is faster than placing blocks one by one.

**Write what you will say in your video.** You can read from it while filming.
