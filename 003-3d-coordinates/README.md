# 🧊 003 — 3D 좌표

8주 동안 평면 격자에서 출발해 좌표 (x, y, z)로 입체 구조물과 픽셀 아트, 그리고 보는 방향에 따라 다르게 보이는 조각상까지 코드로 직접 만듭니다.

2D 격자에서 시작해 높이(y) 축을 더하며 입체 공간 감각을 기르는 **네 번째 코스**입니다.
