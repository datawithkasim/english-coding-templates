# 🧊 003 — 3D 좌표

8주 동안 평면 좌표 (x, y)에서 출발해, 바닥 좌표 (x, z)를 거쳐, 입체 좌표 (x, y, z)까지 익히고, 마지막에는 2D 그림과 3D 조각을 모두 담은 **나만의 미술관**을 코드로 직접 짓습니다.

벽에 점을 찍는 2D 좌표에서 시작해 한 축씩 더해 가며 입체 공간 감각을 기르는 **네 번째 코스**입니다.

> 💡 1~3주는 벽 위 평면(x, y) 좌표에 익숙해지고, 5주차부터 세 축을 모두 쓰는 입체(x, y, z)로 확장합니다 — 조각상 → 엔더 드래곤 → 몹 머리를 직접 따라 만들며 입체 감각을 기릅니다. 8주차에는 배운 모든 것을 모아 미술관을 완성합니다.
>
> 📝 워크시트는 세 단계입니다 — **초급**(1~3학년) · **중급**(3~5학년) · **심화**(5~6학년). 아이 수준에 맞는 단계를 골라 주세요.
>
> 🧩 **확장** 줄은 진도와 별개인 **보너스 도전**입니다. 확장은 3주차 픽셀 아트 뒤, 확장 2는 6주차 입체 만들기 뒤에 더 도전하고 싶은 학생을 위한 자료입니다.
>
> 🔥 **확장 3 (최고 심화)** 는 코스 전체를 마친 뒤 도전하는 가장 어려운 보너스입니다. 블록의 색을 손으로 고르지 않고, **코드가 나눗셈 나머지(15는 3으로도 5로도 나누어짐)로 색을 정하는** 알고리즘을 픽셀 아트로 눈으로 봅니다 — 15칸마다 같은 무늬가 반복되어 세로 줄무늬가 생깁니다.
