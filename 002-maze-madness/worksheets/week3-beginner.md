# 🔗 M002 Week 3 — AND Conditions

**Topic:** AND — two things both true · **Course:** Maze Madness · **Level:** Beginner · **Time:** about 25 minutes

`AND` checks two things. The action runs only when **both** are true.

---

## 1 · Predict 🔮

```
if wall ahead AND wall on right:
    turn left
```

Wall ahead. Right side open. Does it turn? Circle one: **yes · no**


Wall ahead AND wall on right. Does it turn? Circle one: **yes · no**


---

## 2 · Find the Difference 🐛

Turn left only at a corner: wall ahead AND wall on right.

```
# clean
if wall ahead AND wall on right:
    turn left
```

```
# buggy
if wall ahead:
    turn left
```

**When does the buggy one turn too early? Circle one:** at any wall ahead · only at corners · never

---

## 3 · Fill the Gap ✏️

Turn left only when **both** walls are there. Fill the blank.

```
if wall ahead ____ wall on right:
    turn left
```

**Missing word? Circle one:** AND · IF · THEN
