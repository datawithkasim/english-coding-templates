# 🧊 M002 Week 5 — English Worksheet (Beginner)

**Topic:** Mini Cube Puzzle · **Course:** Maze Madness · **Level:** Beginner · **Time:** about 30 minutes

This week you finish the **mini cube**. You already have starter code that builds part of the cube. Keep the same pattern going until the cube is full — then the agent must **break a hole in the top** and climb out.

---

## 1 · Predict 🔮

Read the steps. Before you imagine the agent doing it, circle or write your answer.

```
repeat 5 times:
    place block
    move forward
```

**What does the agent leave behind? Circle one:** a line of blocks · a tower · a hole

**How long is it?**

```
break block above

```

**What does the agent do here? Circle one:** climbs out through a hole · digs down · builds a wall

## 2 · Find the Difference 🐛

Each pair shows clean steps first, then a broken version of the same idea. Circle what's different and write one short sentence about the bug.

**Pair A** — The agent should finish the layer, **then** move up.

```
# clean
place block
place block
place block
move up
```

```
# buggy
place block
move up
place block
place block
```

**What is wrong?**

<div class="write-space short"></div>

**Pair B** — To climb out, the agent must make the hole **first**, then move into it.

```
# clean
break block above
move up
```

```
# buggy
move up
break block above
```

**What is wrong? What does the buggy agent bump into?**

## 4 · Finish the Cube 📸

Open this week's world. You will find the **mini cube starter** — some of the cube is already built.

1. Keep the **same pattern** the starter uses.
2. Repeat it until the **whole cube is filled**.
3. Make sure the agent **breaks a hole in the top** and climbs out.

When you finish, come back here. Send a photo or video of the finished cube (and the hole in the top!), then explain what you did. Use these sentence starters — write 2 or 3 sentences.

> To keep building, I …
>
> I made the hole in the top by …
>
> One tricky moment was when …

## 5 · Record Your Walkthrough 🎥

Take a video on your phone (or a parent's phone) while the agent builds the cube. Talk like you are teaching a friend. Try to use these words: **cube**, **layer**, **pattern**, **hole**.

> 1. Show the starter cube before you run your code.
> 2. Run your code and watch the cube fill up.
> 3. Show the agent breaking the hole in the top and climbing out.

**Write what you will say in your video.** You can read from it while filming.
