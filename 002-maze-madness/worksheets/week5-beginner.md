# 🧊 M002 Week 5 — Mini Cube Puzzle

**Topic:** Mini Cube Puzzle · **Course:** Maze Madness · **Level:** Beginner · **Time:** about 25 minutes

Finish the cube. Fill it, then break the top and climb out.

---

## 1 · Predict 🔮

```
repeat 5 times:
    place block
    move forward
```

**Circle one:** a line · a tower · a hole

```
break block above
move up
```

**Circle one:** climbs out · digs down · builds wall

---

## 2 · Find the Bug 🐛

```
# clean
place block
place block
move up
```

```
# buggy
place block
move up
place block
```

**What is wrong? Circle one:** moves up too soon · places too few · turns wrong

---

## 3 · Fill the Gap ✏️

**Missing line? Circle one:** break block above · place block · turn right
