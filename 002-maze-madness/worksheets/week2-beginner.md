# 🧭 M002 Week 2 — Turning (Beginner)

**Topic:** Turn at Walls · **Course:** Maze Madness · **Level:** Beginner · **Time:** about 25 minutes

Wall ahead? Turn. No wall? Move forward.

---

## 1 · Read It 🔮

```
if wall ahead:
    turn right
otherwise:
    move forward
```

**Wall is in front. What does the agent do? Circle one:** turn right · move forward · stop

**No wall. What does it do? Circle one:** move forward · turn right · stop

---

## 2 · Spot It 🐛

Wall ahead = turn.

```
# clean
if wall ahead:
    turn right
otherwise:
    move forward
```

```
# buggy
if path is clear:
    turn right
otherwise:
    move forward
```

**Circle the line that is wrong.**

---

## 3 · Fill It ✏️

No wall = move forward. One word is missing.

```
if wall ahead:
    turn right
____________:
    move forward
```

**Missing word? Circle one:** otherwise · repeat · while

---

## 4 · Show Your Work 📸🎥

Open your homework world. Turn at walls to get the agent out.

Record **one video** (a phone is fine). Show two things:

**1 · Your code.** Scroll through it. Say what each part does.

**2 · Your build.** Point the camera. Name the parts.

Say these out loud, filling in the blanks:

> Today I built ______.
>
> I built it using this code: ______.
>
> In this code I used ______.
>
> The hardest part was ______.
>
> That part was hard because ______.
