# 🌀 M002 Week 1 — While Loop

**Topic:** Repeat Until a Wall · **Course:** Maze Madness · **Level:** Beginner · **Time:** about 25 minutes

A while loop repeats. It runs until a wall stops it.

---

## 1 · Read It 🔮

```
while no wall ahead:
    move forward
```

**When does the agent stop? Circle one:** after 3 steps · at the wall · never

**Why? Circle one:** a wall · it is tired · time is up

---

## 2 · Match It 🔗

Draw a line.

| Code | Means |
|------|-------|
| while | check the wall |
| no wall ahead | true keeps it going |
| move forward | repeat |

---

## 3 · Fill One Word ✏️

The agent walks until the wall.

```
while ____ wall ahead:
    move forward
```

**Missing word? Circle one:** no · big · one
