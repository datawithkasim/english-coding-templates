# 🌀 002 — Maze Madness / 미로 챌린지

8주 동안 미로 탐색, 레드스톤, 레버 작동을 배워 마지막에 **15×15×15 큐브 미로** 를 완성합니다.
8 weeks of maze navigation, redstone, and lever mechanics — building up to a full **15×15×15 cube maze**.
