# 🤖 AI001 Week 1 — English Worksheet

**Topic:** First Prompt — Meeting Replit Agent · **Course:** AI Coding with Replit Agent · **Time:** about 45 minutes

This week you meet your AI builder, **Replit Agent**. You do not type code. You type a **prompt** — an instruction in plain words — and the AI builds a web page for you.

---

Read each prompt. Before you imagine the AI building it, write what you think will appear on the screen.

## 3 · Try It and Show Me 📸

Open Replit and turn on the Agent. Build a web page with **your own name** on it, using only prompts. Try changing the background color and making the text bigger by writing a second prompt.

When you finish, come back here. Send a **photo or video** of your page, then explain what you did. Use these sentence starters — write 4 to 6 sentences total.

> My first prompt asked the AI to …
>
> When I read the result, I saw …
>
> My second prompt changed …
>
> One word I added to make it clearer was …
>
> Something surprised me when …
>
> If I had more time, I would …
