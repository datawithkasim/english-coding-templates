# 🤖 AI001 Week 5 — English Worksheet

**Topic:** Iteration — The First Result Is Never Final · **Course:** AI Coding with Replit Agent · **Time:** about 45 minutes

The AI's first try is almost never perfect — and that is normal. This week you learn to **iterate**: instead of deleting everything and starting over, you say "change just this one part." Good iteration prompts are specific and often include "keep this part the same."

---

## 3 · Iterate on a Project 📸

Open one of your earlier projects (or build a quick one). Write **several** "change just this part" prompts to raise its quality. After each one, look at the result before writing the next.

When you finish, send a **photo or video** of the improved project, then explain what you did. Use these sentence starters — write 4 to 6 sentences total.

> I started with a project that …
>
> My first change prompt asked the AI to …
>
> I kept one part the same by saying …
>
> One change did not work, so I …
>
> The project got better because …
>
> If I had more time, I would …
