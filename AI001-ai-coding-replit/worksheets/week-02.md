# 🤖 AI001 Week 2 — English Worksheet

**Topic:** Good Prompts vs Bad Prompts · **Course:** AI Coding with Replit Agent · **Time:** about 45 minutes

The AI does what it **reads**, not what you imagine. If your prompt is vague, the AI guesses, and the result is random. A good prompt has three things: it is **specific**, it has a **clear outcome**, and it gives **constraints** (what to do and what not to do).

## 3 · Three Prompts, One Task 📸

Pick **one** task — for example, "a page about my favorite animal". Write it **three ways** and run each in Replit:

1. A **bad** prompt (vague, like "make a page").
2. A **medium** prompt (a bit more detail).
3. A **good** prompt (specific + clear outcome + constraints).

Look at the three results side by side. Send a **photo or video** of the three results, then explain the difference. Use these sentence starters — write 4 to 6 sentences total.

> My task was to build …
>
> My bad prompt was … and the result …
>
> My good prompt added … so the result …
>
> The biggest difference between bad and good was …
>
> A constraint I used was …
>
> Next time I will always remember to …
