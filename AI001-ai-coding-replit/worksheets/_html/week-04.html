🤖 AI001 Week 4 — English Worksheet

Topic: Breaking It Into Steps · Course: AI Coding with Replit Agent · Time: about 45 minutes

If you ask the AI to build a whole app in one giant prompt, it loses focus and the result is sloppy. This week you learn to go first → then → finally. Each step is its own prompt, and the AI builds on top of the last result.

1 · Predict 🔮

Read each plan. Before the AI builds it, write what you think the result will be like.

[One prompt]
"Build a complete to-do app with adding, deleting,
checking off tasks, saving, dark mode, and animations."

This asks for everything at once. What might come out messy or missing?

[Step 1]
"Build a basic to-do screen: an input box, an 'Add' button,
and an empty list."

This is just the first step. What is NOT here yet? Is that okay?

[Step 2]
"Now make the 'Add' button put the input text into the list."
[Step 3]
"Add an 'X' next to each item that deletes it."

Why is building Step 2 then Step 3 easier than asking for both at once?

2 · Spot the Weak Plan 🐛

Each plan below has a step problem. Read what the student wanted, then rewrite the steps in a better order or split them up. Then explain the fix.

Plan A — The steps are in the wrong order: it tries to delete items before any list exists.

[Step 1] "Add a delete button to each item."
[Step 2] "Make a screen with an input box and an empty list."

Hint: you cannot delete from a list that is not built yet.

Write the steps in the right order:

Why was the order wrong?

Plan B — This "step" is actually five things crammed into one prompt.

[Step 1] "Add the list, the delete button, the checkbox,
dark mode, and a save feature."

Hint: split it into separate steps, one feature each.

Write it as separate steps:

Why is one-feature-per-step better?

Plan C — The student forgot the very first step: there is no basic screen to build on.

[Step 1] "Make the add button add text to the list."
[Step 2] "Add a delete button."

Hint: what has to exist before an 'add' button can work?

Write a Step 0 that goes first:

Why does the screen need to come first?

3 · Build an App in Steps 📸

Pick a simple app idea (a to-do list, a score counter, a flashcard page — your choice). Plan it as 4 to 5 steps, then run each step as its own prompt in Replit.

When you finish, send a photo or video of the final app, then explain what you did. Use these sentence starters — write 4 to 6 sentences total.

My app idea was …

My first step built …

My next steps added …

Building one step at a time helped because …

One step I had to redo was …

If I had more time, I would …
