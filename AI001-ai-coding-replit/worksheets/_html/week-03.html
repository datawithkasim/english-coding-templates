🤖 AI001 Week 3 — English Worksheet

Topic: Context and Examples · Course: AI Coding with Replit Agent · Time: about 45 minutes

This week you make your prompts smarter by adding context (who you are, what you like, who it is for) and examples ("make it like this site"). You also use negative constraints — telling the AI what not to do.

1 · Predict 🔮

Read each prompt. Before the AI builds it, write how the extra information will change the result.

Build a gallery site that shows cat photos.

This works, but the AI knows nothing about you. What might it guess wrong?

I'm 12 and I love cats. Build a gallery of cat photos,
Pinterest-style cards, with cute warm colors.

The context and example were added. How will this result be different from the first?

Build a study site for me. Don't use complicated words,
and don't make the buttons tiny.

Two "don't" rules were added. Why might telling the AI what NOT to do help?

2 · Spot the Weak Prompt 🐛

Each prompt below is missing context, an example, or a needed "don't" rule. Read what the student wanted, then rewrite it to add what is missing.

Prompt A — The student is 11 and wants their first project: a simple page about soccer, easy to understand.

Make a soccer page.

Hint: add who you are ("I'm 11, this is my first project") and a "keep it simple" rule.

Write a stronger prompt:

What context or constraint did you add?

Prompt B — The student wants their page to look like the Naver homepage layout.

Make a homepage with news on it.

Hint: give an example: "make the layout like the Naver homepage".

Write a stronger prompt:

What example did you give?

Prompt C — The student keeps getting dark colors and hard words, but they want bright colors and simple words.

Make a fun page about space.

Hint: add the example feel you want ("bright, cartoon-style") and the "don't" rules.

Write a stronger prompt:

What did you add?

3 · A Site About Something You Love 📸

Pick a topic you really like. Write one prompt that includes all three:

  • context — who you are or who the site is for,
  • an example — at least one site or style to copy ("like ___"),
  • a constraint — at least one "don't" rule.

Run it in Replit. Send a photo or video of the result, then explain what you did. Use these sentence starters — write 4 to 6 sentences total.

My topic was …

The context I gave the AI was …

The example I used was "make it like …"

The "don't" rule I added was …

Adding context changed the result by …

If I had more time, I would …

4 · Save Your Best Prompt 🗒️

Write the full prompt you used in part 3, word for word. Underline the context, circle the example, and box the "don't" rule so you can see all three parts.
