🤖 AI001 Week 1 — English Worksheet

Topic: First Prompt — Meeting Replit Agent · Course: AI Coding with Replit Agent · Time: about 45 minutes

This week you meet your AI builder, Replit Agent. You do not type code. You type a prompt — an instruction in plain words — and the AI builds a web page for you.

1 · Predict 🔮

Read each prompt. Before you imagine the AI building it, write what you think will appear on the screen.

Build a simple web page that shows "Hello World".

What will be on the page? Just words, or pictures too?

Build a web page. The title says "Hello World",
the background is light yellow, and the text is dark green.

How is this result different from the first one? What changed?

Make a page.

This prompt is very short. What might go wrong? Will you get exactly what you pictured?

2 · Spot the Weak Prompt 🐛

Each prompt below was meant to get a clear result, but it is too vague or missing something. Read what the student wanted, then rewrite the prompt so the AI knows exactly what to build. After that, explain why the first one was weak.

Prompt A — The student wanted a page that says their name, Minji, in big letters.

Make my name.

Hint: the AI does not know your name, the size, or that you want a web page.

Write a clearer prompt:

Why was the first one weak? Why does yours work better?

Prompt B — The student wanted a page with a blue background and white text that says "Welcome".

Make it blue.

Hint: make what blue? And what should the page say?

Write a clearer prompt:

Why was the first one weak? Why does yours work better?

Prompt C — The student already has a "Hello World" page and now wants the text bigger and the background pink.

Change everything.

Hint: name the two things you want changed.

Write a clearer prompt:

Why was the first one weak? Why does yours work better?

3 · Try It and Show Me 📸

Open Replit and turn on the Agent. Build a web page with your own name on it, using only prompts. Try changing the background color and making the text bigger by writing a second prompt.

When you finish, come back here. Send a photo or video of your page, then explain what you did. Use these sentence starters — write 4 to 6 sentences total.

My first prompt asked the AI to …

When I read the result, I saw …

My second prompt changed …

One word I added to make it clearer was …

Something surprised me when …

If I had more time, I would …
