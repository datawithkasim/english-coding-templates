🤖 AI001 Week 7 — English Worksheet

Topic: Understanding the Code — Don't Just Build · Course: AI Coding with Replit Agent · Time: about 45 minutes

If you use AI-built code without understanding it, you do not grow as a coder — you stay dependent on the AI. This week you learn the most powerful prompt of all: "Explain what each line does." Once you understand a piece of code, you can reuse it anywhere.

1 · Predict 🔮

Read each prompt. Before the AI answers, write what kind of answer you expect to get back.

Just make it work.

Will this prompt teach you anything about how the code works? Why or why not?

Explain what each line of this code does, in Korean,
in words an elementary student could understand.

What kind of answer will this give? How is it different from "just make it work"?

Look at this small piece of code:

let score = 0;
button.addEventListener("click", function() {
  score = score + 1;
  output.textContent = score;
});

In your own words, what do you THINK this code does when you click the button?

2 · Spot the Weak Prompt 🐛

Each prompt below will not help the student actually understand their code. Read what the student wanted, then rewrite the prompt to get a clear, line-by-line explanation.

Prompt A — The student wants to understand every line of a function, in simple words.

What does this do?

Hint: ask it to go line by line, and ask for simple language.

Write a stronger prompt:

Why is "line by line" better than "what does this do"?

Prompt B — The student understood the code, and now wonders if there is a simpler way to write it.

Make new code.

Hint: ask "is there a simpler way to do this, and why?"

Write a stronger prompt:

Why is asking "is there another way?" a good habit?

Prompt C — The student wants the AI to add a comment above each line so they can read it later.

Comment the code.

Hint: say where the comments go (above each line) and what language they should be in.

Write a stronger prompt:

Why do comments help you reuse code later?

3 · Understand Your Own Code 📸

Open one of your projects. Pick a short piece of its code and ask the AI to explain each line. Read the explanation, then write your own short note above each line (in Korean or English) saying what it does.

When you finish, send a photo or video of your commented code, then explain what you learned. Use these sentence starters — write 4 to 6 sentences total.

The piece of code I picked does …

The AI explained that the first line …

A line I did not understand before was …

Now I know it does …

One thing I could reuse this code for is …

A question I still have is …

4 · Teach It Back 🗒️

Read your code out loud, line by line, in your own words — as if teaching someone next to you. Write your spoken explanation here. If you can explain it, you understand it.
