🤖 AI001 Week 2 — English Worksheet

Topic: Good Prompts vs Bad Prompts · Course: AI Coding with Replit Agent · Time: about 45 minutes

The AI does what it reads, not what you imagine. If your prompt is vague, the AI guesses, and the result is random. A good prompt has three things: it is specific, it has a clear outcome, and it gives constraints (what to do and what not to do).

1 · Predict 🔮

Read each prompt. Before the AI builds it, write what kind of result you expect — and how sure you are that you will get what you pictured.

Make a cool web page.

Will every student who types this get the same thing? Why or why not?

Make an about-me page.

This is better than "cool web page". But what is still missing?

Make an about-me page with three sections — name, age,
and hobbies — in pastel colors.

Which of the three rules (specific / clear outcome / constraints) does this prompt use?

2 · Spot the Weak Prompt 🐛

Each prompt below is too weak to give a good result. Read what the student wanted, then rewrite the prompt so it is specific, has a clear outcome, and gives constraints. Then explain what was missing.

Prompt A — The student wanted a calculator with number buttons, the four math buttons, a result screen, and a reset button.

Make a calculator.

Hint: list the buttons and the screen. Add a color theme as a constraint.

Write a strong prompt:

What was missing from the first one?

Prompt B — The student wanted a to-do list page, but written in a way the AI can picture clearly.

Make something to write my tasks.

Hint: say it is a web page, name the parts (input box, add button, list), and describe how it should look.

Write a strong prompt:

What was missing from the first one?

Prompt C — The student wanted a page, but without a dark background and without tiny text (those hurt their eyes).

Make a nice page but not ugly.

Hint: "not ugly" tells the AI nothing. Use real constraints: "don't use a dark background", "don't make the text small".

Write a strong prompt:

What was missing from the first one?

3 · Three Prompts, One Task 📸

Pick one task — for example, "a page about my favorite animal". Write it three ways and run each in Replit:

  1. A bad prompt (vague, like "make a page").
  2. A medium prompt (a bit more detail).
  3. A good prompt (specific + clear outcome + constraints).

Look at the three results side by side. Send a photo or video of the three results, then explain the difference. Use these sentence starters — write 4 to 6 sentences total.

My task was to build …

My bad prompt was … and the result …

My good prompt added … so the result …

The biggest difference between bad and good was …

A constraint I used was …

Next time I will always remember to …
