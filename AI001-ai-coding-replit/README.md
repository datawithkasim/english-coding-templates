# 🤖 AI001 — AI 코딩 with Replit Agent

8주 동안 코드를 직접 타이핑하는 대신 **AI(Replit Agent)에게 말로 시켜서** 자기만의 웹 앱을 만듭니다. 매주 한 가지 "프롬프트 원칙"에 집중하며, 마지막에는 처음부터 끝까지 학생 스스로 설계한 앱을 완성해 발표합니다.
