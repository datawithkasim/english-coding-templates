# 🚀 WD003 — 개인 포트폴리오 사이트

8주 동안 HTML, CSS, JavaScript를 함께 사용해 **자신을 소개하는 포트폴리오 웹사이트**를 기획부터 공개까지 직접 만들어 봅니다. 매주 한 가지 섹션이나 기능을 완성하며, 이전 주차 결과물 위에 차곡차곡 쌓아 올립니다.

> 이 코스는 **영어코딩** Web Dev CSS · Web Dev JavaScript 수업을 마친 학생에게 적합합니다. 매 주 한 가지 핵심 개념에 집중합니다.
