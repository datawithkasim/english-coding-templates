# 🎮 영어코딩 — 숙제 자료실

안녕하세요, 학부모님! 👋

이 페이지는 **영어코딩** 수업의 숙제 자료를 모아둔 곳입니다. 매주 수업이 끝난 뒤 자녀가 사용할 월드 파일과 워크시트를 여기서 받으실 수 있습니다.

---

## 📚 마인크래프트 코스

### 🏗️ [000 — 빌더 기초](./000-builder-basics)

8주짜리 입문 코스입니다. 에이전트 움직이기부터 시작해서 다리, 농가, 우물, 집까지 직접 만들어 봅니다. 코딩이 처음이어도 따라올 수 있도록 구성되어 있습니다.

### 🔺 [001 — 피라미드 문제](./001-pyramid-problems)

8주 코스. 한 줄짜리 블록 길에서 출발해 다이아몬드 구조물까지 코드로 직접 쌓아 올립니다.

### 🌀 [002 — 미로 챌린지](./002-maze-madness)

8주 코스. 미로 탐색과 레드스톤, 레버 작동을 배우며 마지막에는 20×20×20 크기의 큐브 미로에 도전합니다.

### 🧊 [003 — 3D 좌표](./003-3d-coordinates)

8주 코스. 평면 격자에서 출발해 좌표 (x, y, z)로 입체 구조물과 픽셀 아트, 보는 방향에 따라 다르게 보이는 조각상까지 만듭니다.

### 🎮 [004 — 함수와 게임](./004-functions-games)

8주 코스. 함수로 코드를 정리하고 게임 요소와 상호작용하며, 플레이어가 직접 탐험하는 입체 공간과 미니게임을 만듭니다.

### 🎢 [005 — 테마파크 크리에이터](./005-theme-park-creator)

8주 코스. 마인크래프트에 내장된 명령어(/커맨드)와 함수로 게임 가게와 롤러코스터가 있는 나만의 테마파크를 만들고 그랜드 오프닝까지 엽니다.

### ✨ [추가 워크시트](./999-extras)

특정 코스 주차에 묶이지 않은 단독 워크시트 모음입니다. 하나의 개념을 집중 연습하거나 보충·심화가 필요할 때 사용하세요.

---

## 🕹️ 로블록스 코스

로블록스 스튜디오와 텍스트 코딩 언어 **Luau(루아우)** 로 진짜 게임을 만들어 로블록스에 직접 출시하는 트랙입니다. (PC 전용 무료 프로그램입니다.)

### 🏃 [RB001 — 점프맵(옵비) 만들기](./RB001-obby-architect)

8주 코스. 첫 스크립트부터 변수·이벤트·조건문·반복문까지 배우며, 직접 만든 점프맵 게임을 로블록스에 출시합니다.

### 📜 [RB002 — 스토리 퀘스트](./RB002-story-quest)

8주 코스. 말하는 NPC와 갈래 대화, 퀘스트 아이템이 있는 모험 게임을 만듭니다. 문장을 코드로 다루는 코스라 영어 연습 효과가 큽니다.

### 🏭 [RB003 — 타이쿤 엔지니어](./RB003-tycoon-engineer)

8주 코스. 드로퍼·컨베이어·업그레이드가 있는 타이쿤 게임을 만들고, 게임을 꺼도 돈이 남는 저장 기능까지 넣습니다.

### ⚔️ [RB004 — 아레나 레전드](./RB004-arena-legends)

8주 코스. 두 팀이 맞붙는 라운드제 멀티플레이어 미니게임을 만듭니다. 팀, 타이머, 공지, 파워업, 점수판까지.

### 🎮 [RB005 — 드림 게임 스튜디오](./RB005-dream-game-studio)

8주 졸업 작품 코스. 지금까지 배운 모든 것으로 자기만의 오리지널 게임을 기획부터 출시까지 직접 해냅니다.

## 📂 자료 사용법

각 코스 폴더 안에서 아래 자료를 받으실 수 있습니다.

| 자료 | 설명 |
|------|------|
| 📦 `worlds/` | 마인크래프트 월드 파일(`.mcworld`) — 내려받은 뒤 더블 클릭하면 마인크래프트 에듀케이션에서 자동으로 열립니다. (마인크래프트 코스 전용) |
| 📝 `worksheets/` | 매주 영어 워크시트 PDF — **초급**(처음 배우는 친구용)과 **심화**(더 깊이 연습하는 친구용) 두 가지입니다. |
