# 🎮 잉글리시 코딩 — 숙제 자료실

안녕하세요, 학부모님! 👋

이 페이지는 **잉글리시 코딩** 마인크래프트 수업의 숙제 자료를 모아둔 곳입니다. 매주 수업이 끝난 뒤 자녀가 사용할 월드 파일과 코드 템플릿을 여기서 받으실 수 있습니다.

---

## 📚 코스 안내

### 🏗️ [000 — 빌더 기초](./000-builder-basics)

8주짜리 입문 코스입니다. 에이전트 움직이기부터 시작해서 다리, 농가, 우물, 집까지 직접 만들어 봅니다. 코딩이 처음이어도 따라올 수 있도록 구성되어 있습니다.

### 🔺 [001 — 피라미드 문제](./001-pyramid-problems)

8주 코스. 한 줄짜리 블록 길에서 출발해 다이아몬드 구조물까지 코드로 직접 쌓아 올립니다.

### 🌀 [002 — 미로 챌린지](./002-maze-madness)

8주 코스. 미로 탐색과 레드스톤, 레버 작동을 배우며 마지막에는 15×15×15 크기의 큐브 미로에 도전합니다.

---

## 📂 자료 사용법

각 코스 폴더 안에는 **세 개의 하위 폴더** 가 있습니다.

| 폴더 | 설명 |
|------|------|
| 📦 `worlds/` | 마인크래프트 월드 파일(`.mcworld`) — 내려받은 뒤 더블 클릭하면 마인크래프트 에듀케이션에서 자동으로 열립니다. |
| 🐍 `python/` | 파이썬 코드 템플릿(`.py`) — 코드 빌더 화면에 복사해 붙여 넣으세요. |
| 🧱 `blocks/` | 블록 코드 파일(`.mkcd`) — 메이크코드에서 "가져오기"로 불러오시면 됩니다. |
