# 🎮 English Coding — 숙제 자료실 / Homework Hub

안녕하세요, 학부모님! 👋
Welcome, parents! 👋

이 페이지는 **English Coding** 마인크래프트 수업의 숙제 자료를 모아둔 곳입니다. 매주 수업 후 자녀가 사용할 월드 파일과 코드 템플릿을 여기서 받으실 수 있습니다.

This page collects homework files for **English Coding** Minecraft lessons. Download world files and code templates your child needs after each weekly lesson.

---

## 📚 코스 / Courses

### 🏗️ [000 — Builder Basics / 빌더 기초](./000-builder-basics)

8주 첫 코스. 에이전트 이동부터 다리·농장·우물·집까지 직접 만듭니다. 코딩 경험 없어도 OK.
8-week starter course. From agent movement to bridges, farms, wells, and houses. No coding experience needed.

### 🔺 [001 — Pyramid Problems / 피라미드 문제](./001-pyramid-problems)

8주 코스. 첫 피라미드부터 다이아몬드 구조물까지 직접 코드로 만듭니다.
8-week course. Build everything from a single block path to a full diamond using code.

### 🌀 [002 — Maze Madness / 미로 챌린지](./002-maze-madness)

8주 코스. 미로 탐색, 레드스톤, 레버 작동을 거쳐 15×15×15 큐브 미로를 완성합니다.
8-week course. Maze navigation, redstone, and lever mechanics — leading to a 15×15×15 cube maze.

## 📂 자료 사용법 / How to use the files

각 코스 폴더 안에는 **세 개의 하위 폴더** 가 있습니다.
Each course folder has **three subfolders**:

| 폴더 / Folder | 설명 / Description |
|----------------|--------------------|
| 📦 `worlds/` | 마인크래프트 월드 파일 (`.mcworld`) — 다운로드 후 더블클릭하면 마인크래프트 Education에서 자동으로 열립니다 / Minecraft world files — double-click after download to open in Minecraft Education |
| 🐍 `python/` | 파이썬 코드 템플릿 (`.py`) — Code Builder에 복사해서 붙여넣으세요 / Python templates — copy and paste into Code Builder |
| 🧱 `blocks/` | 블록 코드 파일 (`.mkcd`) — MakeCode에서 "가져오기 (Import)"로 불러오세요 / Block code files — use "Import" in MakeCode |
