# English Coding — Curriculum Templates

Public repository of homework template files for [English Coding](https://english-coding.co.uk) Minecraft Education courses.

Parents and students download world files, code templates, and block files for each week's homework directly from here.

## Courses

- [`001-pyramid-problems/`](./001-pyramid-problems) — MS001 / MBS001 — Pyramid Problems (8 weeks → Diamond final)

More courses coming: MS002 / MBS002 — Maze Madness.

## How students use this repo

Each course folder has three subfolders + per-week files:

```
<course>/
├── worlds/        # .mcworld files for Minecraft Education
├── python/        # .py templates for MBS track (Python)
└── blocks/        # .mkcd templates for MS track (Blocks)
```

Click a file → click "Download" (or copy the raw URL).

## How parents share with students

Each week's Notion hub page (linked from [English Coding Operations Hub](https://www.notion.so/English-Coding-Operations-Hub-32ec90549d4181899c4bcd13af7a6745)) contains direct download links for that week's files.
