# 🏗️ 000 — Builder Basics / 빌더 기초

8주 동안 마인크래프트 에이전트로 첫 명령어부터 집·우물·농가가 있는 마을까지 직접 만들어 봅니다.
8 weeks of Minecraft Agent coding — from your very first command to a complete village (house · well · farm house).

이 코스는 **첫 코딩 코스**입니다. 코딩이 처음이어도 괜찮습니다.
This is the **first coding course**. No prior coding experience needed.
