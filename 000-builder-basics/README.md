# 🏗️ 000 — Builder Basics / 빌더 기초

8주 동안 마인크래프트 에이전트로 첫 명령어부터 완성된 부지(다리·농장·우물·집)까지 직접 만듭니다.
8 weeks of Minecraft Agent coding — from your very first command to a complete property (bridge · farm · well · house).

이 코스는 **첫 코딩 코스** 입니다. 코딩 경험이 전혀 없어도 시작할 수 있습니다.
This is the **first coding course**. No prior coding experience needed.

---

## 📅 주차별 학습 / Weekly Plan

| 주 / Week | 주제 / Topic | 결과물 / Result |
|----------|-------------|----------------|
| 1 | 에이전트 이동 / Moving the Agent | 에이전트가 명령대로 움직임 / Agent moves on command |
| 2 | 블록 놓기 / Placing Blocks | 블록 5개 줄지어 놓기 / 5-block line |
| 3 | 다리 만들기 / Bridge | 10블록 다리 / 10-block bridge |
| 4 | 농장 만들기 / Farm | 3×3 농장 / 3×3 farm |
| 5 | 우물 만들기 / Well | 사각형 우물 / Square well |
| 6 | 집 만들기 / House | 5×5×3 집 벽 / 5×5×3 house walls |
| 7 | 집 + 지붕 / House with Roof | 지붕이 있는 집 / House with roof |
| 8 | **최종: 완성된 부지** / **Final: Complete Property** | 다리·농장·우물·집이 모두 있는 부지 / Property with bridge, farm, well, and house |
