# 🏠 M000 Week 5 — English Worksheet (Beginner)

**Topic:** Roofed House · **Course:** Builder Basics · **Level:** Beginner · **Time:** about 30 minutes

---

## 1 · Predict 🔮

Read the steps. Circle your answer.

```
repeat 4 times:
    repeat 4 times:
        place block down
        move forward
    turn right
```

**What shape does this make on the ground? Circle one:** line · square · tower

**Why does the agent turn right after each side? Circle one:** to start the next side · to go higher · to dig down

```
move up by 1
repeat 3 times:
    place block down
    move forward
```

**Do the new blocks land at the same height as before, or higher? Circle one:** same · higher

---

## 2 · Find the Difference 🐛

Clean steps first, then a broken version. Circle the bug.

**Pair A** — The agent should place 5 blocks.

```
# clean
repeat 5 times:
    place block down
    move forward
```

```
# buggy
repeat 4 times:
    place block down
    move forward
```

**What is wrong? Circle one:** only 4 blocks, not 5 · one block too many · a wrong turn

**Pair B** — The agent should build one straight side, **then** turn.

```
# clean
repeat 4 times:
    place block down
    move forward
turn right
```

```
# buggy
repeat 4 times:
    place block down
    move forward
    turn right
```

**What is wrong? Circle one:** it turns after every block · it never turns · it goes up

---

## 3 · Fill the Gap ✏️

The roof goes **on top** of the walls, so the agent must go up first. One line is missing.

```
____________
place block down
move forward
place block down
```

**Which line is missing? Circle one:** move up by 1 · turn right · move forward

---

## 4 · Show Your Work 📸🎥

Switch to your homework world. Build a **small house with a roof** — any size you like.

Record **one video** (a phone is fine). Show two things:

**1 · Your code.** Scroll through it. Say what each part does.

**2 · Your build.** Point the camera. Name the parts.

Say these out loud, filling in the blanks:

> Today I built ______.
>
> I built it using this code: ______.
>
> In this code I used ______.
>
> The hardest part was ______.
>
> That part was hard because ______.
