## 2 · Find the Bug 🐛

Each code block is broken. Write one or two sentences on what is wrong.

**Bug A** — The agent should build a bridge that is exactly **5 blocks long**.

```
repeat 4 times:
    place block down
    move forward
```

**What is wrong, and what number would fix it?**

<div class="write-space"></div>

**Bug B** — The agent should lay 4 blocks in a flat line, next to each other.

```
repeat 4 times:
    place block down
    move up by 1
```

**What is wrong? What should the agent do instead of "move up by 1"?**

<div class="write-space"></div>

**Bug C** — The agent should place a block, then step forward, over and over, to make a bridge.

```
repeat 5 times:
    move forward
    place block down
```

**Hint:** think about the order of the two lines inside the loop.

**What goes wrong when the agent moves before it places? Explain.**
