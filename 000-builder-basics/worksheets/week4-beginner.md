# 🧱 M000 Week 4 — English Worksheet (Beginner)

**Topic:** House Walls (No Roof) · **Course:** Builder Basics · **Level:** Beginner · **Time:** about 30 minutes

---

## 1 · Predict 🔮

Read the steps. Circle your answer.

```
repeat 5 times:
    place block down
    move forward
```

**Is this lying flat on the ground or standing up tall? Circle one:** flat · tall

**What is it? Circle one:** a wall on the ground · a tower

```
repeat 5 times:
    place block down
    move up by 1
```

**Is this lying flat on the ground or standing up tall? Circle one:** flat · tall

**How tall is it? Circle one:** 5 blocks tall · 1 block tall · 3 blocks tall

---

## 2 · Find the Difference 🐛

Clean steps first, then a broken version. Circle the bug.

**Pair A** — The agent should build a wall **5 blocks long** on the ground.

```
# clean
repeat 5 times:
    place block down
    move forward
```

```
# buggy
repeat 4 times:
    place block down
    move forward
```

**What is wrong? Circle one:** only 4 blocks, not 5 · one block too many · a wrong turn

**Pair B** — The agent should build a wall going **up**, not along the ground.

```
# clean
repeat 5 times:
    place block down
    move up by 1
```

```
# buggy
repeat 5 times:
    place block down
    move forward
```

**What is wrong? Circle one:** it goes along the ground, not up · it goes up, not along · a block is missing

---

## 3 · Fill the Gap ✏️

The agent should build one wall, then turn, then build a second wall. One line is missing.

```
repeat 5 times:
    place block down
    move forward
____________
repeat 5 times:
    place block down
    move forward
```

**Which line is missing? Circle one:** turn right · move up by 1 · place block down

---

## 4 · Show Your Work 📸🎥

Switch to your homework world. Build **one wall** for your house — about 5 blocks long and 3 blocks tall.

Record **one video** — one take, no stopping (a phone is fine). Show these in order:

**1 · Your code.** Scroll through it. Say what each part does.

**2 · Your build.** Point the camera. Name the parts.

Say these out loud, filling in the blanks:

> Today I built ______.
>
> I built it using this code: ______.
>
> In this code I used ______.
>
> The hardest part was ______.
>
> That part was hard because ______.
>
> The most fun part was ______.
>
> Something new I learned was ______.
