## 1 · Predict 🔮

```
    move forward
```

## 2 · Find the Difference 🐛

Each pair shows clean steps first, then a broken version of the same idea. Circle what's different and write one short sentence about the bug.

**Pair A** — The agent should build a wall **5 blocks long** on the ground.

```
# clean
repeat 5 times:
    place block down
    move forward
```

```
# buggy
repeat 4 times:
    place block down
    move forward
```

**What is wrong?**

<div class="write-space short"></div>

**Pair B** — The agent should build a wall going **up**, not along the ground.

```
# clean
repeat 5 times:
    place block down
    move up by 1
```

```
# buggy
repeat 5 times:
    place block down
    move forward
```

**What is wrong?**

## 4 · Tell Me What You Built 📸

Now switch to your homework world. Build **one wall** for your house — about 5 blocks long and 3 blocks tall. When you finish, come back here.

Send a photo or video of your wall, then explain what you did. Use these sentence starters — write 2 or 3 sentences.

> My loop did …
>
> My wall is … blocks long and … blocks tall.
>
> The hardest part was …

<div class="write-space tall" style="min-height: 240px"></div>

---

## 5 · Record Your Walkthrough 🎥

Take a video on your phone (or a parent's phone) and show your wall. Talk like you are teaching a friend. Try to use these words: **loop**, **repeat**, **wall**, **block**.

> 1. Show the wall and say how long and tall it is.
> 2. Read your loop out loud.
> 3. Show one bug you hit and how you fixed it.

**Write what you will say in your video.** You can read from it while filming.
