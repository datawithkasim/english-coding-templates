🗿 Builder Basics — The Impossible Statue (Extension · Beginner)

Topic: Place + Move, On and Off · Course: Builder Basics · Level: Beginner · Type: Extension Activity · Time: about 30 minutes

🧩 This is a bonus challenge, not a weekly lesson. Try it once you can place blocks and move the agent.

1 · Read the World 🔍

Open the Impossible Statue world. It is a puzzle. Look at the colours first:

  • 🟥 Red blocks — where the agent starts and where it ends.
  • ⬛ Crying obsidian — build here. Place a block on every dark spot.
  • ⬜ Glass — skip here. Move past it, but place nothing.

What do you do on a crying obsidian spot? Circle one: place a block · skip it

What do you do on a glass spot? Circle one: place a block · skip it

2 · Place On, Place Off 💡

On a build spot, you place. On a skip spot, you only move — no place.

place block down
move forward
move forward
place block down

How many blocks does this place? Circle one: 1 · 2 · 3

Why do we skip a step? Circle one: it is a glass spot · it is a build spot · it is the start

3 · Fill the Gap ✏️

The pattern should be: build, skip, build (B S B). One line is missing.

place block down
move forward
____________
place block down

Which line is missing? Circle one: move forward · place block down · turn right

4 · Find the Mistake 🐛

The pattern should be build, skip, build (B S B). This code places a block on the skip spot by mistake.

place block down
move forward
place block down
move forward
place block down

What is wrong? Circle one: it builds on the skip spot · a build is missing · it turns the wrong way

5 · Show Your Work 📸🎥

Go to the Impossible Statue world. Place blocks on the crying obsidian spots, skip the glass spots, and finish on the red end block.

Record one video (a phone is fine). Show two things:

1 · Your code. Scroll through it. Say what each part does.

2 · Your build. Point the camera. Name the parts.

Say these out loud, filling in the blanks:

Today I built ______.

I built it using this code: ______.

In this code I used ______.

The hardest part was ______.

That part was hard because ______.
