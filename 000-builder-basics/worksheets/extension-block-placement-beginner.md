# 🧱 Block Placement Practice

**Builder Basics · Bonus · about 20 minutes**

> 🧩 The switch is **ON**. The agent draws while it walks!

---

## 1 · Move = Blocks 👀

The switch **place on move** is ON. It stays ON.

| The agent… | It drops… |
|:----------:|:---------:|
| **moves** 🚶 | 🟫 🟫 🟫 blocks |
| **turns** 🔄 | nothing (it stays still) |

**Move = blocks.  Turn = no blocks.**

---

## 2 · Circle It ✏️

`move forward by 3`. Circle one:

# 🟫🟫🟫 3 blocks    ·    ⬜ no blocks

`turn right`. Circle one:

# 🟫 1 block    ·    ⬜ no blocks

`move forward by 2`. Circle one:

# 🟫 1 block    ·    🟫🟫 2 blocks

---

## 3 · Make a Corner 📐

The agent walks, turns, then walks again:

```
move forward by 3
turn right
move forward by 3
```

The trail looks like this:

# 🟫🟫🟫
# ⬜⬜🟫
# ⬜⬜🟫
# ⬜⬜🟫

**Count them. How many blocks? Circle one:** 3 · 6 · 9

**What made the trail bend? Circle one:** `move forward` · `turn right`

**Did `turn right` drop a block? Circle one:** yes · no

---

## 4 · Go Up 🪜

`move up` is a step too. So it drops blocks too!

Walk, up, walk, up = **stairs** 🪜

```
move forward by 1
move up by 1
move forward by 1
move up by 1
```

**Stairs go… circle one:** up ⬆️ · down ⬇️

**How many blocks do these 4 steps drop? Circle one:** 2 · 4 · 8
