# 🏃 RB001 — 점프맵(옵비) 만들기

8주 동안 **로블록스 스튜디오**와 텍스트 코딩 언어 **Luau(루아우)** 로 직접 점프맵(옵비) 게임을 만들어 진짜 로블록스에 출시합니다.

로블록스 트랙의 **첫 번째 코스**입니다. 로블록스 1~4 코스(마인크래프트 코스와 별개 트랙)를 마친 뒤 졸업 작품 코스로 이어집니다.
