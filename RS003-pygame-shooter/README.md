# 🎮 RS003 — 파이게임 스페이스 슈터

16주 과정입니다. Python 기초(변수·리스트·딕셔너리·함수)를 익힌 학생이 **Pygame**으로 실시간 스페이스 슈터 미니 게임을 처음부터 끝까지 직접 만듭니다. 게임 창 열기에서 시작해 우주선 그리기·키보드 조작·적·총알·충돌·점수까지, 매주 만든 코드를 버리지 않고 한 단계씩 쌓아 올립니다.

한 슬라이드 덱을 **학습 주**(새 개념)와 **응용 주**(그 개념을 깊이 활용) 두 주에 걸쳐 사용합니다.
